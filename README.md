# bspwm-dotfiles
Personal .config files for bspwm

Heavily based/inspired from N3k0Ch4n's awm configs (https://github.com/N3k0Ch4n/Another_dotfiles).

### setup
- **WM**: bspwm
- **Terminal**: alacritty
- **Bar**: polybar
- **Compositor**: picom
- **Hotkey**: sxkhd
- **Launcher**: rofi
- **Font**: CaskaydiaCove Nerd Font
